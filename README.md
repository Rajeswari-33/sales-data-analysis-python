# sales-data-analysis-python
Week-2 Internship Project
# Project Overview
This project analyzes sales data using Python to evaluate business performance. It includes
calculations of total price using unit price and quantity, along with product-wise, category-wise,
and monthly sales analysis to generate meaningful business insights.

# Project Objectives
- Create a structured sales dataset
- Calculate total price from unit price and quantity
- Analyze overall sales performance
- Perform product-wise and category-wise analysis
- Analyze monthly sales trends
- Generate a business summary report
- Export the final dataset for reporting

# Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- GitHub

# Dataset Description
The dataset contains 50 sales records with the following fields:
- Order_ID
- Order_Date
- Product
- Category
- Quantity
- Unit_Price
- Total_Price

# Key Analysis Performed
- Total revenue and total order calculation
- Product-wise sales performance analysis
- Category-wise sales distribution
- Monthly sales trend analysis

# Output Files
- Week2_Sales_Data_Analysis.ipynb
- week2_sales_analysis_output.csv

# Conclusion
This project demonstrates how raw sales data can be transformed into useful business insights
using Python, supporting data-driven decision-making.
